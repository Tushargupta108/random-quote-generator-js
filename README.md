 # Random Quote Generator

IN This project -> we will  click a button and  add event listner on that button then it will choose a random value  which will consider as index of array from which it randomly select and index and then event goes on array and showcse  quote wrriten on that particular index in array 

A simple and responsive web application that displays a random quote whenever the user clicks a button.

## Features
- Generate random quotes
- Simple and clean UI
- Responsive design
- Instant quote updates
